# Architecture Website Interface

A simple architecture-themed landing page built with HTML and CSS.

## About the Project
This project was created to practice frontend layout and styling using pure CSS.  
It includes a full-screen banner, a navigation menu with hover effects, and a main section designed to guide users to take action.

## Features
- Full-screen banner layout
- Navigation menu with hover underline effect
- Call-to-action buttons with hover animation
- Clean and minimal interface design

## Technologies
- HTML5
- CSS3

## What I Did
I designed and built the entire user interface on my own.  
The project helped me improve my understanding of layout structure, CSS transitions, and responsive design basics.

## Purpose
This project was developed as a learning and portfolio project.
